# Notes - NLW Expert 📝
 Projeto desenvolvido juntamente com o professor Diego Fernandes da Rocketseat em React durante o evento NLW Expert.

* Responsividade
* Buscar Notas

## Tecnologias utilizadas

* React
* TypeScript
* Tailwind CSS
* Web API's
  - Web Storage
  - Web Speech
